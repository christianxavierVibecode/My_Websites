# 🥣 Landing Page Soto Yogyakarta Cak Sumalik

Landing page profesional, responsif, dan modern untuk **Warung Soto Yogyakarta "Cak Sumalik"** (bisnis kuliner delivery/takeaway lokal). 

Desain menggunakan tema gelap premium dengan aksen oranye hangat, dilengkapi dengan integrasi pesanan langsung ke WhatsApp dan peta lokasi yang disesuaikan secara visual.

## 🏗️ Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript
- **CSS Framework**: Tailwind CSS via CDN (dengan konfigurasi tema khusus)
- **Icons**: FontAwesome v6
- **Typography**: Google Fonts (Poppins & Inter)

## 📋 Fitur Utama

1. **Sticky Navigation Bar**: Menu navigasi yang tetap berada di atas saat di-scroll dengan transisi latar belakang blur dan tombol hamburger responsif untuk perangkat mobile.
2. **Hero Section**: Judul utama promosi yang menarik dengan latar belakang visual estetis dan tombol pintasan menu serta pesanan WhatsApp.
3. **About Section**: Kisah warung Soto Cak Sumalik dan galeri visual suasana warung serta proses penyajian yang bersih.
4. **Menu Section**: Grid responsif berisi hidangan utama:
   - **Soto Ayam** - Rp 25.000 (Best Seller)
   - **Soto Daging** - Rp 35.000
   - **Soto Iga** - Rp 40.000 (Premium)
   Setiap item dilengkapi dengan tombol "Pesan Sekarang" yang mengarah ke chat WhatsApp dengan teks pesan otomatis (pre-filled text) sesuai menu yang dipilih.
5. **Contact & Location**: Informasi alamat, jam operasional, kontak, serta peta Google Maps interaktif yang difilter dengan gaya dark mode agar menyatu dengan palet warna website.
6. **Responsive Layout**: Diuji penuh pada viewport mobile, tablet, dan desktop.

## 🚀 Setup & Cara Menjalankan

### Cara 1: Menggunakan XAMPP (Direkomendasikan karena folder berada di `htdocs`)
1. Pastikan aplikasi **XAMPP** sudah terinstal di komputer Anda.
2. Buka **XAMPP Control Panel** dan jalankan modul **Apache**.
3. Buka peramban (browser) Anda dan akses URL berikut:
   ```text
   http://localhost/hackaton/
   ```

### Cara 2: Membuka Langsung File HTML
1. Buka folder proyek `c:\xampp\htdocs\hackaton\`.
2. Klik dua kali pada file `index.html` untuk membukanya di browser pilihan Anda.
